símbolo

Grano

La M se cosecha de una rejilla 5×5 de granos de maíz: muchas unidades modulares que componen una sola identidad. Es mono-oro; el azul maíz se reserva para UI y acentos. Respeto mínimo = 1 módulo (un grano). Tamaño mínimo: 16 px.

Reglas

sí

  • Respeta el área de un módulo alrededor del logo.
  • Usa oro para la marca y cal para texto sobre oscuro.
  • Reserva Space Mono para CLI, código y labels.

no

  • No recolorees el símbolo fuera de la paleta.
  • No deformes, rotes ni añadas sombras o efectos.
  • No uses azul maíz en el logo (es para UI).
